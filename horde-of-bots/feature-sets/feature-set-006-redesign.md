# [feature-set-006] Brutalist redesign (SimplifyOps style)

## Goal
Replace the current friendly-Inter board look with the SimplifyOps brutalist editorial system: dark `#0a0a0a` background, single rust `#c4724a` accent, hairline borders, Big Shoulders Display headlines + Inter body + JetBrains Mono metadata. The board should feel like the same product family as `simplifyops.com`.

## Rationale
The user owns simplifyops.com and wants change-mate visually unified with it. The styleguide is well-defined and complete. Redesign happens before the kill-Supabase work (feature-set-005) because visual changes get faster feedback and the Supabase removal doesn't touch CSS — the two streams are independent.
