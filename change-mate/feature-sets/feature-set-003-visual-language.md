# [feature-set-003] Visual Language

## Goal
Give change-mate a distinctive, delightful visual identity — one that makes the board feel alive and rewards progress visually.

## Rationale
Workflow hygiene (feature-set-002) is about ticket mechanics; visual language is about how the board looks and feels when someone uses it. Keeping these streams separate prevents polish work from crowding out mechanics work (or vice versa). The set starts with animated forge scenes (CM-011); over time it's likely to grow with clickable relationship chips, dependency graph views, color themes, and card-move animations.
